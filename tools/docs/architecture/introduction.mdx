## Technologies Used - Overview
Here are listed some of the specific technologies used for the implementation of the project:

- Authentication: JSON Web Tokens - JWT
- Databases - Persistence: MongoDB, PostgeSQL
- Testing: JEST
- External Communication Protocols: REST, gRPC (deprecated)
- Frameworks: ΝestJS
- PubSub technology: NATS
- Message Streaming Technology: JetStream by NATS
- Container Technology: Docker
- Tracing-Observability: Jaeger, Grafana
- Metrics: Prometheus
- API Gateway - Proxy: Envoy
